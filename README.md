#### 安装方式（开发模式安装）
 - 把下载后的.crx扩展名的离线Chrome插件的文件扩展名改成.zip或者.rar
 - 解压压缩文件
 - 在Chrome的地址栏中输入：chrome://extensions/ 打开Chrome浏览器的扩展程序管理界面，并在该界面的右上方的开发者模式按钮上打勾
 - 在勾选开发者模式选项以后，在该页面就会出现加载正在开发的扩展程序等按钮，点击“加载正在开发的扩展程序”按钮，并选择刚刚解压的Chrome插件文件夹的位置
